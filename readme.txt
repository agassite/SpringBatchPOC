Bonjour,

Pour executer le batch, vous devez préalablement créer une table dans
une base de données de votre choix, avec le script sql 'create_table_personne.sql' situé
dans le répertoire script.
Ensuite je vous conseille de faire un buid des sources via Maven (mvn clean install), puis d'executer la classe 'BatchPersonne'.
